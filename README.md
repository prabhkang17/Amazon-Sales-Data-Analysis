# Amazon Sales Data Analysis

## Project Overview
This project analyzes Amazon sales data using Python. The objective is to identify product trends, customer ratings, discount patterns, and category performance through exploratory data analysis and visualization.

## Objectives
- Analyze Amazon product data.
- Identify top-rated products.
- Study discount percentages.
- Examine category performance.
- Generate business insights using data visualization.

## Tools and Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains:
- Product Name
- Category
- Actual Price
- Discounted Price
- Discount Percentage
- Rating
- Review Count

## Project Workflow

### 1. Data Collection
Loaded Amazon sales dataset in Google Colab.

### 2. Data Cleaning
- Removed missing values
- Checked duplicates
- Converted prices and discounts into numerical format

### 3. Exploratory Data Analysis
- Product rating analysis
- Category analysis
- Discount analysis
- Price analysis

### 4. Data Visualization
- Bar Charts
- Histograms
- Pie Charts
- Scatter Plots

### 5. Business Insights
- High discounts attract customer attention.
- Most products have ratings between 3.5 and 4.5.
- Some categories dominate product listings.
- Premium products often maintain good ratings.

## Conclusion
This project demonstrates how Python can be used for real-world business data analysis. Through data cleaning, visualization, and insight generation, meaningful conclusions were extracted from Amazon sales data.
